📋 Tabel Absen Sederhana

Project ini merupakan aplikasi web sederhana untuk mencatat kehadiran menggunakan HTML, CSS, dan JavaScript.

---

🚀 Fitur Utama

- Menampilkan tabel daftar nama
- Dropdown untuk memilih status kehadiran:
  - Hadir
  - Izin
  - Sakit
  - Alpa
- Tombol Simpan untuk memproses data
- Menampilkan jumlah total kehadiran berdasarkan kategori
- Tampilan menarik dengan background light blue
- Gambar dekorasi di sisi kiri dan kanan

---

🛠️ Teknologi yang Digunakan

- HTML5 → Struktur halaman
- CSS3 → Styling dan tampilan
- JavaScript → Interaksi dan logika program

▶️ Cara Menjalankan

Menggunakan Browser / Live Server

1. Buka folder project di VS Code
2. Klik kanan "index.html"
3. Pilih Open with Live Server

---

Menggunakan Laragon

1. Simpan folder project di:
   C:\laragon\www\
2. Jalankan Laragon (Start All)
3. Buka browser dan akses:
   http://localhost/nama-folder

---

💡 Contoh Penggunaan

1. Pilih status kehadiran pada setiap nama
2. Klik tombol Simpan
3. Hasil jumlah kehadiran akan muncul di bawah tabel
